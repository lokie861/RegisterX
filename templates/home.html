Welcome to PLC Modbus Converter

A simple tool to convert PLC register addresses into Modbus format quickly and accurately.

Start Converting

Features

  • Instant register to Modbus conversion
  • Supports multiple PLC types
  • One-click copy to clipboard
  • Easy bug reporting

How it Works

  1. Select your PLC type from the dropdown.
  2. Enter the register address (e.g., M2401).
  3. Click Convert to see the Modbus mapping.
  4. Copy results with one click.

Latest Updates

  • v1.1 – New copy-to-clipboard UI
